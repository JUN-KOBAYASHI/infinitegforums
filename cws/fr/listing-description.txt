Après avoir installé l'extension, ajustez-la à vos besoins en cliquant sur son icône dans le menu des extensions pour accéder à la page des options. En effet, les fonctionnalités autres que le défilement infini sont désactivées par défaut.

Vous pouvez consulter la liste de toutes les améliorations ici : https://gerrit.avm99963.com/plugins/gitiles/infinitegforums/+/HEAD/docs/features.md

Les nouveaux ajouts dans les futures mises à jour seront désactivés par défaut, et pourront être activés manuellement dans la page des options. Rejoignez https://groups.google.com/g/twpowertools-discuss pour recevoir des mises à jour sur les nouvelles fonctionnalités.

Le code source est disponible ici : https://gerrit.avm99963.com/plugins/gitiles/infinitegforums. Les contributions sont les bienvenues !
